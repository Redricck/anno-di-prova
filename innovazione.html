Script Python per la didattica

Una piccola toolchain in Python (con python-docx) che genera i documenti delle prove: testi dei compiti nelle tre file A/B/C, simulazioni, recuperi e fogli di valutazione — con impaginazione coerente e zero copia-incolla manuale.

genera_compito.py — generazione delle tre file
for fila in ["A", "B", "C"]:
    doc = crea_documento(intestazione, regolamento)
    for n, esercizio in enumerate(esercizi[fila], 1):
        aggiungi_esercizio(doc, n, esercizio, punti[n])
    doc.save(f"Compito_Fila_{fila}.docx")  # + versione PDP allineata

"Biblioteca di Quartiere"

Il database didattico costruito per il percorso SQL della 4S: sei tabelle collegate (autori, libri, lettori, eventi, presentazioni, prestiti) su LibreOffice Base/HSQLDB, distribuito alle classi con le esercitazioni.

$ cat schema.sql

Lo schema del database didattico

Biblioteca di Quartiere — schema logico
Autori        (id_autore, nome, cognome, nazionalità)
Libri         (id_libro, titolo, anno_pubblicazione, genere)
Lettori       (codice_fiscale, nome, cognome, indirizzo)
Eventi        (id_evento, data_evento, orario_evento, sala, id_autore→Autori)
Presentazioni (id_evento→Eventi, id_libro→Libri, tema)
Prestiti      (codice_fiscale→Lettori, id_libro→Libri, data_inizio, data_restituzione)
-- data_restituzione NULL = prestito ancora aperto
